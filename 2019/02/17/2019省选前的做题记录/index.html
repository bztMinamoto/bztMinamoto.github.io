LG4948 数列求和
数论。
推个式子。结果式 $\sum_{i=1}^{n} (n^k\ast a^n)$。好像直接不知道怎么搞。感觉这就是一个等比数列的样子，那就用证明等比数列通项公式的办法证明这道题。
令$T(p)=\sum_{i=1}^{n} (n^p\ast a^n)$。
$(a-1)\ast T(p)=n^p\ast a^{n+1}-a+\sum_{i=2}^{n} ((i-1)^p-i^p)\ast a^i$
而$(i-1)^p-i^p=\sum_{j=0}^{p-1} C(p,j)\ast i^j\ast (-1)^{p-j}$
然后把这个带进去，再交换一下$\sum$的顺序，就会得到结果式为
$(a-1)\ast T(p)=n^p\ast a^{n+1}-a+\sum_{j=0}^{p-1} (-1)^{p-j}\ast C(p,j)\ast (\sum_{l=2}^{n} l^j*a^l)$
后面那个东西刚好是$T(j)-a$。所以递推$k$就好了。

LG3765 总统选举
平衡树+线段树。
首先要会处理众数的方式，其实也很简单。稍微解释一下。比如说定义一个$sum$为0，$las$为上一个数，若新进来的数不等于$las$，则$sum–$。若$sum==0$，则$las$赋值为0，否则$las$不变。这样下去，$las$就是众数了。这题就用这个思想，用线段树去维护区间的众数。但这题要求的是大于区间长度除以2，所以平衡树维护一下区间个数就好了。

[POI2012]SZA-Cloakroom

[BJWC2018]词韵

考虑建出反串Trie树后，怎么DP。对于一种情况，情况中的点必须是结束点。然后这个点可以向它是结束点的儿子/父亲/兄弟跳一步。儿子对父亲显然，那么就考虑一下父亲对儿子的贡献就可以了。至于兄弟，可以两个兄弟拿出来合并一下。

CF123D String
